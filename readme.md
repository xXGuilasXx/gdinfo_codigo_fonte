# GDINFO Eletrônicos - Catálogo Digital e Sistema de Orçamentos

## Projeto Integrador Extensionista - Responsive Web Developer

Sistema WEB desenvolvido para a empresa **G.d. Info Solucoes em Tecnologia (GDINFO)**, com o objetivo de criar um catálogo digital interativo e um sistema integrado de solicitação de orçamentos.

### Autores
- Guilherme Amaral Cardoso
- Juliano Boaventura

### Tecnologias Utilizadas
- **HTML5** — Estrutura semântica das páginas
- **CSS3** — Estilização responsiva com Flexbox, CSS Grid e Media Queries
- **JavaScript (Vanilla)** — Interatividade, filtros de produtos, validação de formulários e máscaras de input

### Funcionalidades
- Página inicial com banner institucional e estatísticas
- Catálogo de produtos com filtros interativos por categoria
- Listagem de serviços de assistência técnica
- Página institucional (Sobre Nós) com história e valores
- Formulário de solicitação de orçamento com validação e máscara de telefone
- Design totalmente responsivo (Desktop e Mobile)

### Estrutura do Projeto
```
gdinfo-site/
├── index.html          # Página principal (SPA com seções)
├── css/
│   └── style.css       # Estilos e responsividade
├── js/
│   └── script.js       # Lógica e interatividade
├── img/                # Screenshots e imagens do projeto
└── README.md           # Este arquivo
```

### Como Executar
1. Clone o repositório:
   ```bash
   git clone https://github.com/SEU_USUARIO/gdinfo-site.git
   ```
2. Abra o arquivo `index.html` diretamente no navegador, ou utilize um servidor local:
   ```bash
   cd gdinfo-site
   python3 -m http.server 8080
   ```
3. Acesse `http://localhost:8080` no navegador.
